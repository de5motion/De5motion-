# 🎬 Movie Bot

Telegram bot that delivers movies with subscription verification.

## Features
- ✅ Subscription check
- 🎥 Movie delivery
- 📊 User statistics
- 🎯 Interactive menu

## Commands
- `/start` - Show menu
- Enter movie code - Get movie
